在虚拟机中运行OpenWRT

发表于 2020-03-03 分类于 OpenWRT

安装所需软件

虚拟机 ：Workstation Player

img转换为vmdk：StarWind V2V Converter

下载OpenWRT固件镜像

https://downloads.openwrt.org/releases/19.07.2/targets/x86/64/openwrt-19.07.2-x86-64-combined-ext4.img.gz

然后解压获得 openwrt-19.07.2-x86-64-combined-ext4.img

把 img 转换为 vmdk

打开 StarWind V2V Image Converter 软件

选择 Local file 然后点 Next

选择刚才解压的 openwrt-19.07.2-x86-64-combined-ext4.img 文件

选择 Local file

选择 VMDK

选择 VMware Workstation growable image

转换后的 openwrt-19.07.2-x86-64-combined-ext4.vmdk 保存路径

创建虚拟机

打开 VMware Workstation Player 创建新虚拟机

完成后打开上图的位置的文件夹

将刚才转换好的 openwrt-19.07.2-x86-64-combined-ext4.vmdk 复制进去

配置虚拟机

选中刚才创建的虚拟机，然后右键打开设置

移除硬盘

点击下方的“添加” ，然后添加一个硬盘

SCSI

使用现有虚拟磁盘

选择刚才复制进去的 openwrt-19.07.2-x86-64-combined-ext4.vmdk

保持现有的格式

将网卡设置为 桥接模式

然后再添加一个新的网卡，同样也是设置为 桥接模式

确定退出

OpenWRT 配置 IP地址

开机后按回车键显示命令行

输入 

ifconfig br-lan 192.168.0.123

192.168.0.123 是我的内网地址
需要改为你内网没有被使用的地址
例如 192.168.1.123

浏览器输入 192.168.0.123 登录到管理界面

默认密码为：admin

打开 Network - Interfaces

然后Edit LAN

修改 IPv4 address，然后 Save

点击 Save & Apply 保存和应用

OpenWRT-查找管理IP地址

发表于 2020-03-03 分类于 OpenWRT

简介

OpenWRT默认管理地址为：192.168.1.1
如果使用第三方定制的固件，那么地址可能被修改
如果不知道管理地址，可以通过以下方法获取管理地址

通过CMD查找

按住 Ctrl + R 键
输入 cmd 然后按回车健

输入 ipconfig | more 然后按回车健

默认网关地址就是OpenWRT管理地址了
